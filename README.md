# vue-test

> A Vue.js project

## Build Setup

``` bash
# install dependencies
yarn install

# serve in browser with hot reload at localhost:8080
yarn run dev:browser

# serve in electron with hot reload at localhost:8080
yarn run dev:electron

# serve in android (emulator by default) with live reload
yarn run dev:android

# build for production for browser with minification
yarn run build:browser

# build for production for electron with minification
yarn run build:electron

# build for production for android with minification
yarn run build:android

# run on browser
yarn run browser

# run on electron
yarn run electron

# run on android emulator
yarn run android

# run on android device
yarn run android:device
```
